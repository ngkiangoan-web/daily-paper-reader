## 摘要
对纵向文本中的维度情感建模，需要区分当前情感估计与未来情感变化预报。现有方法通常将每个文本视为独立观测，并对两个任务采用类似假设，而未检验它们是否依赖不同的信息源。本文利用纵向自评生态短文和感受词条目研究这一区别。我们提出了特质-状态情感预测（TSAP）框架及其时间扩展E-TSAP用于逐文本的效价和唤醒度预测，并在来自91名用户的1737个条目的保留预测测试集上进行评估。我们进一步提出了情感变化预报混合模型（ACF-Hybrid）用于下一步情感变化预报，并在来自46名用户的保留预报测试集上进行评估。对于预测，E-TSAP在效价上达到0.670的组合皮尔逊相关系数，在唤醒度上达到0.449。对于预报，文本表示的表现不如紧凑的数值轨迹基线：包含文本的模型在效价上仅达到r=0.316，在唤醒度上达到r=0.284，而简单的前一状态基线分别达到r=0.615和r=0.670。ACF-Hybrid使用特定维度的数值轨迹特征，在效价上达到r=0.659，在唤醒度上达到r=0.658。这些结果表明，文本语义支持当前情感预测，而未来情感变化则通过先前的数值轨迹动态更好地捕捉。

## Abstract
Modeling dimensional affect in longitudinal text requires distinguishing current affect estimation from future affective change forecasting. Existing approaches often treat each text as an independent observation and apply similar assumptions to both tasks, without testing whether they rely on different information sources. This paper investigates that distinction using longitudinal self-reported ecological essays and feeling-word entries. We propose the Trait--State Affective Prediction (TSAP) framework and its temporal extension E-TSAP for per-text valence and arousal prediction, evaluated on a held-out prediction test set of 1,737 entries from 91 users. We further propose the Affective Change Forecaster Hybrid (ACF-Hybrid) for next-step affective change forecasting, evaluated on a held-out forecasting test set of 46 users. For prediction, E-TSAP achieves composite Pearson correlations of 0.670 for valence and 0.449 for arousal. For forecasting, textual representations perform worse than compact numeric trajectory baselines: the text-inclusive model achieves only r=0.316 for valence and r=0.284 for arousal, whereas a simple prior-state baseline reaches r=0.615 and r=0.670, respectively. ACF-Hybrid, using dimension-specific numeric trajectory features, achieves r=0.659 for valence and $r=0.658$ for arousal. These results show that textual semantics support current affect prediction, whereas future affective change is better captured through prior numeric trajectory dynamics.

## 一、论文的核心问题与整体含义（研究动机和背景）

本文关注纵向文本（如日记、自我反思等随时间重复收集的自述文本）中的维度情感建模问题。现有方法通常将每个文本视为独立观测，并用相似假设处理两个相关但不同的任务：**当前情感预测**（估计用户当下的效价和唤醒度）与**未来情感变化预报**（预测下一次观测时情感状态的改变量）。然而，这两类任务可能依赖不同的信息源——前者主要依靠文本语义，后者更依赖用户过往的情感轨迹。作者指出，当前研究缺乏在统一框架下系统比较两者信息源贡献的工作，因此提出探究“情感预测与情感预报是否依赖相同的主导信息源”，并设计相应模型与实验来回答这一问题。

## 二、论文提出的方法论：核心思想、关键技术细节

### 2.1 总体框架
论文将纵向情感建模拆分为两个子任务：
- **情感预测**：给定当前文本 \(x_{u,i}\) 和用户身份 \(u\)，估计当前情感状态 \(\hat{\mathbf{y}}_{u,i} = (\hat{v}_{u,i}, \hat{a}_{u,i})\)。
- **情感预报**：给定历史情感轨迹 \(\mathcal{H}_{u,i} = \{(v_{u,j}, a_{u,j})\}_{j=1}^i\)，预测下一步变化 \(\hat{\Delta}_{u,i} = (\hat{\Delta}^{(v)}_{u,i}, \hat{\Delta}^{(a)}_{u,i})\)。

### 2.2 情感预测：TSAP 和 E-TSAP
- **TSAP（Trait–State Affective Prediction）**：
  - 文本编码器：使用微调的 DistilBERT 获取上下文表示，通过掩码均值池化得到特征向量 \(h_{\text{text}} \in \mathbb{R}^{768}\)。
  - 用户嵌入：为每个用户学习一个 128 维可学习向量 \(h_{\text{user}}\)，未见过用户使用共享 UNK 嵌入。
  - 融合：拼接文本表示和用户嵌入，经过共享 MLP 得到潜在表示 \(z\)，再通过两个独立回归头分别预测效价和唤醒度。
  - 损失函数：联合均方误差。
- **E-TSAP（Extended TSAP）**：在 TSAP 基础上增加时间特征流，包括：
  - 滞后特征：前一时刻的 \((v_{i-1}, a_{i-1})\)。
  - 滚动均值特征：最近至多 3 个时间点的均值。
  - 所有时间特征严格因果构建（避免未来信息泄露）。
  - 最终拼接 \(h_{\text{text}}\)、\(h_{\text{user}}\) 和 \(h_{\text{temporal}}\) 进行预测。

### 2.3 情感预报：ACF-Hybrid
**ACF-Hybrid（Affective Change Forecaster Hybrid）** 由两个维度专用的预测器组成：
- **ACF-Residual（用于效价）**：
  - 先使用 Ridge 回归拟合线性趋势 \(\hat{\Delta}^{(v)}_{\text{linear}}\)。
  - 再用一个小型 MLP 学习残差（非线性校正）。
  - 最后进行后验方差校准（\(\alpha_v = 1.15\)）。
- **ACF-Temporal（用于唤醒度）**：
  - 扩展基础时间特征（二阶滞后、加速度、滚动统计、方向一致性指标等）。
  - 通过两层 MLP 预测唤醒度变化。
- 最终输出：效价预测来自 ACF-Residual，唤醒度预测来自 ACF-Temporal。

预报任务中不使用文本编码器和用户嵌入，仅依赖数值轨迹特征。

## 三、实验设计

### 3.1 数据集
- 使用 **SemEval-2026 Task 2** 纵向自评情感数据集，包含 4,501 条记录（2,764 训练 + 1,737 测试），来自 137（训练）和 91（测试）名用户。每个条目包括用户 ID、文本、时间戳、自我报告的效价（-2~2）和唤醒度（0~2）。
- 两种文本类型：叙述性日记和感受词列表。

### 3.2 数据划分
- **预测**：排除少于6条观测的用户后得到 2,677 条；每用户按时间排序，前 80% 为训练，后 20% 为验证；保留未见用户作为验证集。测试集 1,737 条独立保留。
- **预报**：从 2,764 条构造 2,627 个相邻变化标签；每用户最后一个标签作为验证（137 条），其余为训练（2,490 条）。测试集为 46 名用户各一条目标。

### 3.3 对比方法
- **预测基线**：GlobalMean, UserMean, MovingAvg (k=3), Hybrid (MA+UM), TF-IDF–Ridge, DistilBERT-Ridge, DistilBERT-Regressor, DistilBERT-LSTM, 以及提出的 TSAP 和 E-TSAP。
- **预报基线**：RandZero, Linear-BERT, Linear-BERT+Prev, Linear-Prev, 以及提出的 ACF 系列变体（ACF 文本包含版、ACF-Numeric、ACF-Numeric+、ACF-AR、ACF-Residual、ACF-Temporal、ACF-Hybrid）。

### 3.4 评估指标
- 主要指标：**综合皮尔逊相关系数** \(r_{\text{comp}}\)（Fisher-z 平均组内和组间相关），以及 MAE。
- 报告组内、组间分解表现。

## 六、论文的主要结论与发现

1. **情感预测与情感预报依赖不同信息源**：预测任务主要依赖文本语义（DistilBERT 达到 r_comp≈0.66），预报任务中文本表示效果差（r≈0.3），而数值轨迹基线（Linear-Prev）达 r≈0.62~0.67。
2. **TSAP/E-TSAP 在预测上优于纯文本基线**：TSAP 改善了组内跟踪，E-TSAP 进一步提升了效价综合相关至 0.670，但增益主要来自组间结构，组内改善有限。
3. **ACF-Hybrid 在预报上综合最优**：结合 ACF-Residual（效价）和 ACF-Temporal（唤醒度），达到效价 r=0.659、唤醒度 r=0.658，优于文本包含模型和简单线性基线。
4. **效价与唤醒度具有不同统计特性**：效价组间变异更大（ICC=0.327），与文本极性关联强；唤醒度组内变异占主导（ICC=0.164），更依赖时间动态，预报中扩展时间特征更有用。
5. **去除时间特征或用户信息的影响不对称**：预测中去文本效果显著，去用户嵌入损失较大；预报中去文本是最大提升（+0.365/-0.206），去除用户嵌入也有微小正收益（+0.041/+0.021）。

## 七、优点

1. **任务区分明确**：首次在统一框架下系统比较情感预测与情感预报，揭示信息源差异。
2. **模型设计简洁有效**：TSAP 融合文本+用户，E-TSAP 加时间特征，ACF-Hybrid 采用维度专用预报器，体现了任务特化和维度特异性。
3. **评估分解细致**：采用组内/组间相关分解，避免了仅看总体相关而忽略个体动态的错误。
4. **全面的消融实验**：分别测试了文本、用户、时间特征、目标函数等的影响，深入揭示了每个组件的作用。
5. **误差分析深入**：分析了预测误差在不同情感极性和唤醒度水平上的分布，指出了模型在极端情感和稀有状态下的预测困难。

## 八、不足与局限

1. **数据规模小且领域窄**：仅 137 名训练用户和 46 名预报测试用户，且均为英语美国服务行业工人，泛化性差。
2. **时间不规则性处理粗糙**：虽然数据集时间间隔极不规律（分钟到周），但模型仅使用滞后和滚动均值，未考虑实际时间间隔，缺乏时间衰减或连续时间编码。

4. **预报低估极端变化**：所有预报模型预测方差小于真实方差，对罕见的大幅变化预测不足。
5. **未利用额外信号**：仅使用了文本、用户 ID 和数值轨迹，未利用时间间隔、行为元数据、多模态信号等。
6. **统计显著性不足**：预报测试集仅 46 个样本，小模型性能差异可能不显著。
7. **未进行跨数据集验证**：仅在一个数据集上实验，结论的可靠性需更多验证。
